欢迎阅读 jian.css 的文档！
=========
**jian.css** 是一个超轻量级的，现代的 CSS 框架。

它拥有以下特点：

- 🎈️超级轻量，比 water.css、simple.css 等“轻量级” CSS 框架小得多，大小是 simple.css 的 **37%** （main.css）。
- ✔高兼容性，支持大多数主流浏览器。
- 🌙适配深色模式，让网站在黑暗中也能适应环境。
- ✨️效果出众，丝毫不亚于 Bootstrap 等“重量级”框架。
- 🎨️高自定义性，只用一行 JavaScript 代码，将颜色自由切换（使用普通版本）。
- 📏️使用黄金比例，让您的网站更易阅读。
